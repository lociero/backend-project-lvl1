# How to install and use:  
```
$ git clone git@github.com:lociero/backend-project-lvl1.git
$ cd backend-project-lvl1
$ make install
$ make publish
$ sudo npm link
$ brain-even ### brain-even / brain-calc / brain-gcd / brain-progression
ENJOY =)
```
