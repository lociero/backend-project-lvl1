# How to install and use:  
```
$ git clone git@github.com:lociero/backend-project-lvl1.git
$ cd backend-project-lvl1
$ make install
$ make publish
$ sudo npm link
```
### Install and brain-progression game launch:  
```
$ brain-progression
```
### BrainEven game:
```
$ brain-even
```
### BrainGcd game:
```
$ brain-gcd
```
### BrainPrime game:
```
$ brain-prime
```
### BrainCalc game:
```
$ brain-calc
```
